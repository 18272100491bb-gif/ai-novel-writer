# 情感渲染技法

## 情绪递进公式

### 通用的情感节奏
```
铺垫（70%）→ 引爆（10%）→ 余韵（20%）
```

### 催泪三步
1. **建立联结**：让读者喜欢这个角色（至少3000字的日常/互动）
2. **给予希望**：角色即将达成目标前的瞬间
3. **撕碎希望**：用角色的关键选择打破一切

### 燃点构建
1. **压抑蓄力**：让读者和主角一起憋屈
2. **临界点**：一个无法再忍的事件
3. **爆发**：情绪和力量同时释放
4. **回声**：事后周围人的反应（承认/震撼/改观）

## 氛围营造

### 用环境说话
| 场景 | 环境描写关键词 |
|------|--------------|
| 悲伤 | 阴天、雨、暗色调、冷风、落叶 |
| 紧张 | 噪音、快速心跳、影子拉长、空调声 |
| 温暖 | 暖光、小空间、食物香气、缓慢移动 |
| 神秘 | 雾气、回声、光晕、半透明、脚步空荡 |
| 压抑 | 压迫感、门缝、低语、滴水声、影子 |

### 感官调用顺序
> 视觉 > 听觉 > 触觉 > 嗅觉 > 味觉

先在视觉建立场景，再用声音强化氛围，然后触觉代入，最后用气味和味道留下印记。

## 不油腻的情感写法

- **不说"感动"**：写角色喉结动了动、眼眶发酸、声音变哑
- **不说"愤怒"**：写拳头攥得指节发白、呼吸变粗、青筋暴起
- **不说"悲伤"**：写沉默、写发呆、写照常吃饭时眼泪掉进碗里
- **不说"爱情"**：写注意细节、写下意识保护、写记得随口提过的事
