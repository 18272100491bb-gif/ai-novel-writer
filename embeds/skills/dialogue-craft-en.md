# Dialogue Craft

## Character Voice Differentiation

### By Role
- **Strong**: Few words, concise, calm tone
- **Wise**: Metaphors, rhetorical questions, trailing half-sentences
- **Brash**: Short sentences, profanity, strong body language
- **Noble**: Formal speech, proper etiquette, indirect
- **Villain**: Flattery, doublespeak, blame-shifting

### By Emotion
- **Anger**: Short sentences, prominent subjects
- **Fear**: Repetition, stuttering, topic-changing
- **Sadness**: Slower speech, ellipses, avoiding the point
- **Joy**: Talkative, faster pace, exclamations
- **Contempt**: Rhetorical questions, sneers, drawn-out tones

## Subtext Techniques

### Dialogue Iceberg
- Spoken words = only 20% of true thoughts
- Remaining 80% shown through expressions/actions/silence

### Info Density
- **High**: Every line advances plot/reveals info
- **Medium**: Mixed with casual chat
- **Low**: Pure atmosphere/emotion (bonding through banter)

### Subtext Patterns
1. **Non-answer** → Avoiding → "Nice weather today"
2. **Topic change** → Evading → "Did you eat yet?"
3. **Over-detailing** → Guilty conscience
4. **Silence** → Agreement/defiance
5. **Scoffing** → Rejecting engagement
6. **Stammering** → Hesitancy/stress
