# 对话打磨技法

## 角色声线差异化

### 按身份
- **强者**: 话少、用词简洁、语气平缓
- **智者**: 比喻多、反问多、总留半句
- **莽夫**: 短句、脏话、肢体动作强
- **君子**: 书面语、礼节周全、不直说
- **小人**: 逢迎、话里有话、擅长推锅

### 按情绪
- **愤怒**: 句子变短、主语突出（"我杀了你！"）
- **恐惧**: 重复、吞吞吐吐、转移话题
- **悲伤**: 说话变慢、用省略号、避开重点
- **喜悦**: 话多、语速快、用感叹词
- **不屑**: 反问多、冷笑、拉长音

## 潜台词技巧

### 对话冰山
- 角色说的话只占内心想法的 20%
- 剩下的 80% 用表情/动作/沉默来暗示

### 信息密度
- **高密度**：每句话推进剧情/揭示信息（庭审、解密、对峙）
- **中密度**：夹杂日常和闲笔（日常对话、感情戏）
- **低密度**：纯气氛/情绪（废话但建立情感联结）

### 六种潜台词模式
1. **答非所问** → 不想回答 → "今天天气不错"
2. **转移话题** → 回避 → "对了你吃过没？"
3. **过度详细** → 心虚 → 解释太多反而不自然
4. **沉默** → 默认/抗拒 → 停顿就是答案
5. **反问** → 拒绝接话 → "关你什么事？"
6. **重复** → 犹豫/强调 → "我…我真的不知道"

## 对话中的节奏变化

```
交锋  短句快节     ← 双方都在攻/防
暧昧  长句慢节     ← 试探、犹豫
爆发  极短句+粗口  ← 情绪顶点
和解  中长句回暖   ← 情绪回落
```

## 避免的对话陷阱
- ❌ 角色 A 解释给角色 B 听读者已经知道的事
- ❌ 所有角色用一样的说话方式
- ❌ 长篇独白不打断
- ❌ 对话全是「说」辅以大量修饰词
- ✅ 用「放屁」「滚」「嗯」来代替描述
