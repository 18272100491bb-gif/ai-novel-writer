# 人物塑造技法

## 人物弧光三要素

### 起点 → 终点
1. **起点**：角色的初始状态（缺点/恐惧/渴望）
2. **转变触发**：一个无法忽视的事件
3. **终点**：经历后的改变（克服/堕落/觉醒）

### 弧光类型
- **成长型**：懦弱→勇敢（经典英雄之旅）
- **堕落型**：正义→偏执（悲剧路线）
- **觉醒型**：迷茫→找到目标（主角常见）
- **循环型**：克服了A缺点，暴露了B缺点（更真实）

## 行为动机构建

### 三层动机
1. **表层**：角色自己说出来的理由
2. **中层**：角色自己都不完全意识到的理由
3. **深层**：驱动的根本欲望（认可/权力/归属/复仇）

### 实用技巧
- 每个重要决定回答：如果不这么做会怎样？
- 角色为什么是这个反应？用过去的经历解释
- 主角每3-5章展现一个新侧面
- 配角的动机不必完全交代，留下神秘感

## 关系张力四象限

| 象限 | 关系类型 | 张力来源 |
|------|---------|---------|
| 合作/信任 | 师徒、挚友 | 理念分歧、信任考验 |
| 合作/怀疑 | 盟友、合作者 | 利益冲突、立场不同 |
| 敌对/仇恨 | 仇敌、死对头 | 直接冲突、互相伤害 |
| 敌对/暧昧 | 亦敌亦友 | 立场对立但互相欣赏 |

## 配角存活原则

1. **三章法则**：新配角出场三章内必须展示一个独特的价值（技能/信息/立场）
2. **工具人检测**：如果这个角色只做一件事（送情报/挡刀/解说），说明是工具人
3. **死亡价值**：如果配角必须死，死因应该是其动机和性格的必然结果
4. **关系网**：每个配角至少和两个其他角色有可持续发展的关系
