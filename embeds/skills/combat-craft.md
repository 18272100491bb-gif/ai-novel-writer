# 战斗描写技法

## 力量层次感的呈现

### 展示 vs 说明
- **避免**："他很强"
- **展示**：余波震碎了百米外的山崖，对手的护体金光像蛋壳一样裂开，裂纹在虚空中蔓延

### 层级递增
1. **基础攻击** → 碰撞声、光芒、冲击波
2. **中等对决** → 地形改变、天象异变
3. **顶级对决** → 规则显现、时空扭曲、法则碰撞
4. **超越描写** → 无法直视、无法理解、只能感受

## 镜头语言五法

1. **远景** → 战斗全景、双方站位、环境破坏
2. **中景** → 动作交互、招式的碰撞
3. **近景** → 表情细节、咬牙、汗水、眼神
4. **特写** → 一个动作慢放（0.5秒拉成5句）
5. **上帝视角** → 旁观者反应、天地异象

## 节奏结构

```
起（10%）  : 互相试探 → 心理博弈
承（30%）  : 小招碰撞 → 逐渐加码
转（40%）  : 底牌掀开 → 陷入危机
合（20%）  : 爆发翻盘 → 余波收尾
```

## 战斗中的对话

- **战斗中** → 短句、断句、用感叹号和省略号
- **对峙时** → 心理攻防、信息战
- **分出胜负后** → 长篇对话揭示动机/世界观

## 差异化描写（不重复"轰"）

| 攻击类型 | 描写词汇 |
|---------|---------|
| 拳脚 | 崩裂、炸响、骨裂声、气浪翻涌 |
| 剑气 | 撕裂虚空、银芒一闪、剑意森然 |
| 法术 | 符文流转、灵光交织、天地失色 |
| 神识 | 无形镇压、灵魂刺痛、意识沉陷 |
| 法宝 | 宝光冲霄、镇压万物、气机锁定 |
