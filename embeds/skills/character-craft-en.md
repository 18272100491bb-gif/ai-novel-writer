# Character Craft

## Three Elements of Character Arc

### Start → End
1. **Start**: Initial state (flaw/fear/desire)
2. **Trigger**: An unavoidable event
3. **End**: Change after experience (growth/fall/awakening)

### Arc Types
- **Growth**: Coward → Brave (classic hero's journey)
- **Fall**: Righteous → Paranoid (tragedy)
- **Awakening**: Lost → Found purpose (common for MC)
- **Cycle**: Overcomes flaw A, exposes flaw B (realistic)

## Motivation Layers

1. **Surface**: What the character says
2. **Middle**: What the character half-knows
3. **Deep**: Core drive (recognition/power/belonging/revenge)

## Relationship Tension Matrix

| Quadrant | Type | Tension Source |
|----------|------|---------------|
| Trust | Mentor, best friend | Ideological split, trust test |
| Suspicion | Ally, collaborator | Conflict of interest |
| Hostility | Rival, nemesis | Direct conflict |
| Ambiguity | Frenemy | Opposing sides, mutual respect |
